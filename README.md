1) 데이터 소개
- 이 대시보드는 한국 제조업 위기를 주제로 통계 자료 3가지를 분석했다. 제조업 생산량 데이터(2020년 1월-2026년 3월)를 통해 각 지역의 공장들이 물건을 얼마나 만들었는지를 알 수 있었고, 2020년을 기준(100)으로 잡고, 그보다 많이 생산하면 100 이상, 적게 생산하면 100 이하로 표시했다. 제조업 취업자 수 데이터(2020년-2025 하반기)를 통해서는 제조업 분야에서 일하는 사람이 얼마나 되는지 알 수 있고, 수출입 무역 데이터(2021년 3월-2026년 3월 누계)는 한국이 어떤 물건을 얼마나 수출하고 수입했으며 그 결과로 돈을 얼마나 벌었는지를 보여준다.

3) 데이터 클린징 과정
- 날짜 오류 수정 : 데이터가 없는 날짜를 조회하면 IndexError가 발생하는 문제가 있어서 ORDER BY 연월 DESC LIMIT 1, MAX() 등을 활용해 DB에서 항상 최신값을 자동으로 가져오도록 수정했다.

*변경 전
SELECT 생산지수 FROM production_index WHERE 연월 = '2026.03'
*변경 후
SELECT 생산지수 FROM production_index
WHERE 지역='전국' AND 산업='제조업'
ORDER BY 연월 DESC LIMIT 1

- 숫자 단위 변환 : 숫자가 너무 길면 읽기 힘들기 때문에 차트 4는 단위를 달러에서 백만 달러로 변환했고, KPI카드는 달러에서 십억 달러로 변환하여 표시했다.

- 인사이트 문구 수정 : 코드를 고치면서 실제 데이터와 맞지 않는 설명 문구도 함께 다듬었다. 예를 들어 '2026년에도 회복을 못했다'는 표현이 사실과 달라 '2025년 하반기에 고점을 회복했지만 월별 등락은 여전히 크다'로 고쳤다.

3) 시각화 결과

차트 1: 전국 제조업 생산지수 추이 (2020–2026)
- 전국 제조업 생산지수를 월별로 나타낸 꺾은선 차트이다. x축은 연월, y축은 생산지수(2020=100 기준)이고, 2023년 구간에 붉은 음영을 깔아서 침체 시기를 강조했다. 그리고 전체 기간 평균값은 노란 점선으로 표시했다.
- 인사이트 : 전국 제조업 생산지수는 2021년 말~2022년 초에 약 122까지 올라갔다. 그러나 2023년에 100~103으로 약 15% 떨어졌다. 이는 전 세계 반도체 수요가 급감하고, 코로나 이후 중국 경기가 기대만큼 살아나지 않은 게 주요 원인이었다. 2024년 말부터 빠르게 회복되어 2025년 하반기에는 고점을 넘어섰지만, 월별 등락이 여전히 커서 안정적인 회복이라고 보기는 어렵다. 회복이 특정 달에 집중되고 저점도 반복되는 패턴은 구조적 불안정이 아직 해소되지 않았음을 시사한다.

차트 2: 지역별 제조업 생산지수 변화 (2020→2025 평균)
- 지역별 2025년 평균 생산지수를 가로 막대 차트로 나타냈다. 생산지수가 115 이상인 지역은 녹색, 100~115는 노란색, 100 미만은 빨간색으로 색을 다르게 해서 어느 지역이 잘하고 있는지 한눈에 보이게 했다.
인사이트 : 2025년을 기준으로 경기와 인천은 생산지수가 145가 넘는 반면, 서울은 유일하게 100 아래로 떨어졌다. 경기와 인천에는 반도체 및 디스플레이 공장이 집중되어 있는 반면, 서울은 공장이 거의 없는 서비스 중심 도시로 바뀌었기 때문이다. 같은 수도권이지만 제조하는 곳과 서비스를 제공하는 지역이 나뉘어진 형태라고 볼 수 있다.

차트 3: 전국 제조업 취업자 수 추이 (반기별)
- 제조업 취업자 수를 6개월 단위로 나타낸 세로 막대 차트이다. 4,400천명 기준선을 노란 점선으로 그어서, 그 아래로 떨어진 시기는 빨간색, 위는 파란색으로 구분했다.
- 인사이트 : 제조업 일자리는 2022년 하반기 약 4,525천 명을 정점으로 2025년 하반기에 4,364천 명까지 161천 명이 줄었다. 같은 기간에 생산지수는 어느 정도 회복되었지만 고용은 따라오지 못한 모습이다. 공장 자동화나 AI의 도입으로 사람 대신 기계가 그 자리를 채우면서, 경기가 좋아져도 일자리는 돌아오지 않는 구조가 됐다.

차트 4: 주요 수출입 품목별 수출액·수입액 비교 (2021.03–2026.03 누계)
- 수출액 기준 상위 12개 품목의 수출액과 수입액을 나란히 세로 막대 차트로 비교했다. 파란색이 수출액, 빨간색이 수입액이고 단위는 백만 달러이다.
- 인사이트 : 수출은 반도체(전기기기), 자동차, 기계류 세 품목에 집중되어 있고, 수입은 원유, 가스와 같은 에너지가 가장 크다. 반도체 하나가 흔들리면 전체 무역수지도 같이 흔들리는 구조이며 실제로 2023년 반도체 불황 시기에 무역적자가 크게 확대되었다. 특정 품목 의존도가 높다는 것은 외부 충격에 그만큼 약하다는 뜻이다.

차트 5: 품목별 무역수지 — 흑자 vs 적자 Top 8
- 화면을 좌우로 나눠서, 왼쪽엔 무역흑자 상위 8개 품목, 오른쪽엔 무역적자 상위 8개 품목을 가로 막대 차트로 보여줬다. 흑자는 녹색, 적자는 빨간색으로 표시했다.
- 인사이트 : 전기기기, 자동차, 선박에서 흑자를 내지만, 광물성연료(원유·가스 포함) 수입에서 압도적인 적자가 나고 있다. 제조업에서 아무리 흑자를 내도 에너지 가격이 오르면 금방 상쇄되는 구조로 2022년 러시아-우크라이나 전쟁 때에도 무역적자로 전환됐다. 에너지 자립 없이는 제조업 흑자도 언제든 위협을 받을 수 있다는 게 핵심 위기이다.
